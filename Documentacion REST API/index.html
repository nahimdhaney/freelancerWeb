Usuario

Estos son los servicios relacionados con los usuarios en la plataforma. Podés insertar, actualizar, eliminar o seleccionarlos.

Test

Una petición de prueba, útil para verificar la conexión con los servicios REST API.

GET
/api/usuario/test

Ejemplo de respuesta

{
  "success": true,
  "message": "Hello earth!"
}

Login

Login.

Atributo Descripción
user el nombre de usuario
password la contraseña
POST
/api/usuario/login/

Ejemplo de petición

{
    "user": "rpaz",
    "password": "1234"
}

Ejemplos de respuesta

{
  "success": true,
  "message": "Ingreso correcto",
  "response": {
    "id": 1,    
    "fullName": "Ricardo Paz",
    "user": "rpaz",
    "password": "1234",
    "email": "rickypazd@icloud.com",
    "enabled": true,
    "type": 1,
    "precio": 600.00
  }
}
{
  "success": false,
  "message": "El usuario no existe"
}
{
  "success": false,
  "message": "Contraseña incorrecta"
}

Registro de usuario

Para registrar un usuario en la plataforma.

Atributo Descripción
fullName el nombre completo del usuario
user el nombre de usuario con el qué ingresar a la plataforma
password la contraseña
email el correo del usuario
type el tipo de usuario (1: freelancer, 2: contratista)
POST
/api/usuario/register/

Ejemplo de petición

{
  "fullName": "Ricardo Paz",
  "user": "rpaz",
  "password": "1234",
  "email": "rickypazd@icloud.com",
  "type": 1
}

Ejemplos de respuesta

{
  "success": true,
  "message": "Registro exitoso",
  "response": {
    "id": 1,  
    "fullName": "Ricardo Paz",
    "user": "rpaz",
    "password": "1234",
    "email": "rickypazd@icloud.com",
    "enabled": true,
    "type": 1,
    "precio": 600.00
  }
}
{
  "success": false,
  "message": "El nombre de usuario ya está siendo ocupado"
}
{
  "success": false,
  "message": "Hubo un error al registrar el usuario"
}
{
  "success": false,
  "message": "El nombre de usuario/correo ya está siendo ocupado"
}

Cambiar contraseña

Para cambiar la contraseña de una cuenta. La idea es utilizar éste servicio con una sesión iniciada en la plataforma

Atributo Descripción
user el nombre de usuario
password la contraseña
newPassword la nueva contraseña
POST
/api/usuario/changePassword/

Ejemplo de petición

{
    "user": "rpaz",
    "password": "1234",
    "newPassword": "v0laf1l3"
}

  "success": false,
  "message": "Contraseña incorrecta"
}

  "success": false,
  "message": "Hubo un error al cambiar la contraseña"
}

Proyecto

Estos son los servicios relacionados con los proyectos en la plataforma. Podés insertar, actualizar, eliminar, seleccionarlos, buscar, etc.

Crear proyecto

Para crear un proyecto en la plataforma.

Atributo Descripción
name el nombre del proyeto
description la descripción del proyecto
category la categoría del proyecto
price el precio del proyecto
date la fecha de creación del proyecto
ownerId el id del contratista a cargo del proyecto
POST
/api/proyecto/insertar/

Ejemplo de petición

{
  "name": "Fidalga app",
  "description": "Para evitar los incendios en las sucursales",
  "category": "Mobile",
  "price": 7000.00,
  "date": "2018-07-14 15:00:00",
  "ownerId": 1
}

Ejemplos de respuesta

{
  "success": true,
  "message": "Proyecto creado",
  "response": {
    "id": 1,
    "name": "Fidalga app",
    "description": "Para evitar los incendios en las sucursales",
    "category": "Mobile",
    "price": 7000.00,
    "date": "2018-07-14 15:00:00",
    "ownerId": 1
  }
}
{
  "success": false,
  "message": "Hubo un error al insertar el proyecto"
}

Actualizar proyecto

Para actualizar un proyecto de la plataforma.

Atributo Descripción
id el id del proyecto
name el nombre del proyeto
description la descripción del proyecto
price el precio del proyecto
date la fecha de creación del proyecto
start (opcional) la fecha de comienzo (por parte del freelancer) en un proyecto
end (opcional) la fecha de finalización del proyecto
ownerId el id del contratista a cargo del proyecto
freelancerId (opcional) el id del freelancer encargado del proyecto
category la categoría del proyecto
POST
/api/proyecto/actualizar/

Ejemplo de petición

{
  "id": 8,
  "name": "Fidalga app",
  "description": "App para evitar el incendio en las sucursales",
  "price": 14000,
  "date": "2018-07-13 00:00:00",
  "start": null,
  "end": null,
  "ownerId": 33,
  "freelancerId": 0,
  "category": "TI, paginas web y software"
}

Ejemplos de respuesta

{
  "success": true,
  "message": "Proyecto actualizado"
}
{
  "success": false,
  "message": "Hubo un error al actualizar los datos del proyecto"
}

Solicitud de proyecto

Estos son los servicios para las solicitudes de proyectos. Se puede realizar una solicitud de parte de un freelancer a un contratista, un contratista puede ver las solicitudes que tiene para un proyecto, un contratista puede aceptar una solicitud, un freelancer puede postularse/despostularse a un proyecto, un freelancer puede confirmar el trabajo para un proyecto.

Solicitar proyecto

Para solicitar trabajar en un proyecto (de parte un freelancer)

Atributo Descripción
projectId el id del proyecto
freelancer el id del freelancer
oferta el precio por el cuál el freelancer se ofrece para trabajar
POST
/api/solicitud/insertar/

Ejemplo de petición

{
  "projectId": 10,
  "freelancerId": 34,
  "oferta": 500
}

Ejemplos de respuesta

{
  "success": true,
  "message": "Solicitud creada",
  "response": {
    "id": 1,
    "projectId": 10,
    "freelancerId": 34,
    "oferta": 500
  }
}
{
  "success": false,
  "message": "Hubo un error al insertar la solicitud de proyecto"
}

Comentario

Estos son los servicios relacionados con los comentarios en la plataforma.

Comentar

Para insertar un comentario en la plataforma. Los comentarios son parte de la interacción entre un freelancer y un contratista.

Atributo Descripción
mensaje el mensaje
fecha la fecha y hora del comentario
usuarioId el id del usuario
solicitudProyectoId el id de la solicitud de proyecto
POST
/api/comentario/insertar/

Ejemplo de petición

{
  "mensaje": "Hola, comenzamos el lunes",
  "fecha": "2018-07-15 15:20:00",
  "usuarioId": 34,
  "solicitudProyectoId": 3,
}

Ejemplos de respuesta

{
  "success": true,
  "message": "Comentario creado",
  "response": {
    "id": 1,
    "mensaje": "Ok, comenzamos el lunes",
    "fecha": "2018-07-15 15:20:00",
    "usuarioId": 34,
    "solicitudProyectoId": 3
  }
}
{
  "success": false,
  "message": "Hubo un error al insertar el comentario"
}
